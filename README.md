## General info
This project is an Android Application allowing users to login via google and store daily/weekly tasks, ticking them off as they go along and seeing a breakdown of goal completion for the time period. Data stored via SQL in a Microsoft Access hosted Database.

	
## Technologies
Project is created with:
* Java
* Android Studio
* SQL
* HTML/CSS
